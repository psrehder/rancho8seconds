# Placar de Dominó

App estático para controle de partidas e pontuação de dominó.

Funcionalidades:
- cadastrar jogadores
- registrar partidas entre dois jogadores
- ver pontuação total por jogador
- exportar/importar backup JSON
- gerar link de compartilhamento para outro aparelho

Coloque a imagem de fundo na mesma pasta como `background.jpg`.

Abra `index.html` em um navegador ou publique como site estático.
